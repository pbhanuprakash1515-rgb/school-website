Project Overview

This project is a static School Website developed as a mini project to demonstrate fundamental web development skills. The website provides essential information about a private school, including school details, gallery, and contact information. It is designed with a simple and user-friendly interface.

Features

Home page with school introduction

About page describing school vision and mission

Gallery page displaying school images

Responsive navigation menu

Clean and structured layout

Cross-browser compatibility

Technologies Used

HTML5

CSS3

JavaScript (basic validation)

Project Structure

Multiple HTML pages for better organization

External CSS for styling

Image gallery using relative paths

Deployment

The website is hosted publicly using GitHub Pages, making it accessible online and suitable for showcasing as a portfolio project.

Learning Outcomes

Understanding of website structure and navigation

Practical experience with HTML, CSS, and JavaScript

Hands-on experience in deploying a website on GitHub Pages

Improved UI/UX and project organization skill
